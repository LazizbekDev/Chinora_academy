## PROJECT DESCRIPTION
(Use this text for the description field - ~1500 characters)

Developed a modern, fully-responsive single-page landing page for Chinora Fashion Academy, a sewing and fashion design education platform. Built with React 19, Vite, and Tailwind CSS, featuring a comprehensive 3-tier course structure with interactive course modules, pricing plans, instructor profiles, and FAQ sections.

**Key Features:**
• Interactive accordion-based course content system with expandable modules
• Multi-tier pricing section with payment modal and card number copying functionality
• GSAP-powered scroll-triggered animations and smooth hero section animations
• Professional instructor profile section with achievements and social media stats
• Responsive FAQ section with contact information
• Image lazy loading with blur placeholders for optimal performance
• Mobile-first responsive design optimized for all devices

**Technical Implementation:**
Utilized React 19, Vite build tool, Tailwind CSS 4, GSAP animations, Radix UI components, React Router, and React Query. Implemented custom hooks, reusable components, and performance optimizations including code splitting and lazy loading.

The website successfully communicates the academy's value proposition and provides a seamless user experience from course discovery to enrollment, with modern design reflecting the professional nature of the fashion academy.

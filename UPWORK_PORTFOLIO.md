# Upwork Portfolio Entry - Chinora Fashion Academy Website

## Project Title
**Modern Responsive Landing Page for Fashion Academy - React/Next.js with Advanced Animations**

---

## Project Description

Developed a modern, fully-responsive single-page landing page for Chinora Fashion Academy, a sewing and fashion design education platform. The website showcases a comprehensive 3-tier course structure (Beginner, Intermediate, and Advanced levels) with interactive course modules, pricing plans, instructor profiles, and student testimonials.

### Key Features Implemented:

**1. Dynamic Course Content System**
- Interactive accordion-based course modules with expandable sub-modules
- Three distinct course levels with detailed curriculum breakdown
- Visual progress indicators and course duration displays
- Collapsible content sections for better user experience

**2. Pricing & Payment Integration**
- Multi-tier pricing structure with visual pricing cards
- Payment modal with integrated card number copying functionality
- Responsive pricing grid with highlighted "Most Popular" plan
- Original vs. discounted price displays
- Payment method selection (Click, Payme, Uzcard, Humo)

**3. Instructor Profile Section**
- Professional instructor bio with achievements showcase
- Social media statistics integration (Instagram, YouTube, Telegram)
- Animated achievement badges and certifications
- Lazy-loaded instructor images with blur effects
- Scroll-triggered animations using GSAP

**4. Interactive FAQ Section**
- Accordion-based frequently asked questions
- Contact information integration (phone, Telegram, email)
- Payment method information display

**5. Advanced Animations & UI/UX**
- GSAP-powered scroll-triggered animations
- Smooth hero section entrance animations with elastic effects
- Hover effects and micro-interactions throughout
- Gradient backgrounds and modern card designs
- Responsive navigation with smooth scrolling

**6. Performance Optimizations**
- Image lazy loading with blur placeholders
- React Query for efficient data management
- Code splitting and component optimization
- Mobile-first responsive design approach

### Technical Highlights:
- Built with React 19 and Vite for optimal performance
- Tailwind CSS 4 for modern, utility-first styling
- Radix UI components for accessible UI elements
- React Router for seamless navigation
- Custom hook implementations for reusable logic
- Toast notifications for user feedback
- Fully responsive across all device sizes
